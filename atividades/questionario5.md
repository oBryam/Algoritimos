6. Qual número inteiro é impresso pelo programa a seguir?

```
Algoritmo "semnome"
Var
  i, s: inteiro
Inicio
  s <- 0
  para i de 0 ate 100 faca
    s <- s + i
  fimpara
  escreval (s)
Fimalgoritmo
```

* **Observação:** A resposta correta é `5050`.
